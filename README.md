# S&P 500 Stock Data Explorer (2013–2018)

An interactive dashboard for exploring closing price trends and five-year
performance statistics across 505 S&P 500 constituent stocks.

## Live Dashboard

[View the dashboard here](https://richardfranklinbao.github.io/data555_sp500_dashboard/Final_Project_3a_flex.html)

## Data Source

[S&P 500 Stock Data — Kaggle](https://www.kaggle.com/datasets/camnugent/sandp500)

The dataset contains approximately 619,040 daily trading records covering 505
S&P 500 constituents from February 2013 to February 2018, collected via
automated scraping of historical price data from public financial data providers.

## Tools

R, flexdashboard, plotly, DT

## Project Importance

Individual stock performance within the S&P 500 varies far more than index-level
returns suggest, and identifying high-volatility or underperforming names early
has direct implications for portfolio risk management. This dashboard enables
investors, analysts, and researchers to quickly explore five years of price
history and comparative performance metrics without requiring any coding expertise.
